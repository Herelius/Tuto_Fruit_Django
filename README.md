# Installation

## Linux

```shell
git clone https://github.com/Herelius/Tuto_Fruit_Django.git
cd Tuto_Fruit_Django
python -m venv .env
source .env/Scripts/activate
pip install -r requirements.txt
```

## Windows

```shell
git clone https://github.com/Herelius/Tuto_Fruit_Django.git
cd Tuto_Fruit_Django
python -m venv .env
.\.env\Scripts\activate
pip install -r requirements.txt
```
